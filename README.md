# Lifelogging Retrieval based on Sematic Concepts Fusion

## Requirements

+ python 2.7

+ gensim

+ sklearn

+ OpenCV

## Structure

### Database Folder

Storing pre-trained model, inverted index, ...

### SRC Folder

+ config.py: Saving global path, threshold, ...

+ word2vec.py: Word processing

+ detect_shot.py: Shot detection
